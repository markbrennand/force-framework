AsynchronousSweeper

global with sharing class AsynchronousSweeper implements Schedulable
Description

Schedulable wrapper that triggers the Asynchronous framework's Sweeper run.

The Sweeper is responsible for housekeeping tasks such as: - Restarting or re-queuing eligible jobs - Advancing jobs stuck in transitional states - Performing framework maintenance as needed

This class is intended to be invoked by a Salesforce cron schedule. The execute method simply delegates to AsynchronousV1.startSweeper(), which enqueues a lightweight job (AsynchronousScheduler.RunSweeper) to perform the actual sweep logic asynchronously.

See README for full details of the Asynchronous API.
